## 🗒️简介说明

******

> 
> 爬取主机域名并自动转换IP/IP段
>
> 2022.7.17

1. 一键采集活跃网站 / IP
2. 域名批量转IP
3. IP段生成
4. 去重复工具


******

## 🎲使用说明

```shell
git clone https://github.com/zsuroy/ipact
pip install -r requirements.txt
python3 ip.py
```

### 😇生成模版
> 开发了包括但不限于以下模版
> 
> 去重复文件以 `-only` 文件名标记
> 

```text
[域名模版]
domain.com
domain2.com

[域名转IP]
domain.com 127.0.0.1
domain2.com 127.0.0.2

[IP模版]
127.0.0.1
127.0.0.2

[IP段模版]
127.0.0.1/24
127.0.0.2/24
```
